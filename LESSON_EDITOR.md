# 📚 Lesson Editor - Полная документация

## Обзор

**Lesson Editor** - это полнофункциональный редактор содержимого уроков для преподавателей в WebLab CMS. Позволяет создавать красиво оформленные уроки с поддержкой Markdown синтаксиса и реальным пред­про­смо­тром.

---

## 🎯 Основные возможности

✅ **Две вкладки интерфейса:**
- 📝 **Редактор** - для редактирования содержания и метаданных урока
- 👁️ **Предпросмотр** - для просмотра как будет выглядеть контент для студентов

✅ **Полная поддержка Markdown:**
- Заголовки всех уровней (h1-h6)
- Текстовое форматирование (жирный, курсив, комбо)
- Списки (упорядоченные и неупорядоченные)
- Ссылки и изображения
- Таблицы с границами
- Блоки кода с подсветкой синтаксиса
- Цитаты и горизонтальные линии

✅ **Подсветка кода для популярных языков:**
- JavaScript/TypeScript

- HTML/CSS
- И 100+ других языков

✅ **Удобные поля формы:**

- Тип урока (Теория/Практика/Видео/Тест)
- Содержание (Markdown)
- Порядок в главе
- Примерное время прохождения

✅ **Управление статусом:**
- Кнопка публикации/скрытия урока
- Статус "Опубликован" или "Черновик"
- Информация о дате создания и обновления

## 📄 Компоненты

### 1. LessonEditorPage
**Файл:** `/app/(authenticated)/admin/courses/[courseId]/chapters/[chapterId]/lessons/[lessonId]/page.tsx`

Главная страница редактора. Загружает урок, отображает breadcrumbs и содержит:
- Загрузку данных (курс, глава, урок)
- Обработка ошибок и состояния загрузки
- Встроение компонентов редактора и справки

**Использует:**
- `useParams()` для получения ID из URL
- `useRouter()` для навигации
- Сервисные функции для загрузки данных

### 2. LessonEditorContent
**Файл:** `/components/admin/lesson-editor-content.tsx`

Основной компонент с двумя вкладками:

#### Editor Tab
Форма редактирования с полями:
- Title (название)
- Description (описание)
- Type (тип урока)
- Content (содержание Markdown)
- Order (порядок)
- EstimatedMinutes (время)

Функциональность:
- Валидация через Zod schema
- Save button с отключением при отсутствии изменений
- Cancel button для отмены
- Автоматическое обновление при сохранении

#### Preview Tab
Реальный предпросмотр:
- Заголовок урока
- Описание (если заполнено)
- Отрендеренный Markdown контент
- Информация о типе и времени

**Props:**
```typescript
interface LessonEditorContentProps {
  courseId: string;      // ID курса из URL
  chapterId: string;     // ID главы из URL

```

**Состояние:**
```typescript
- isSaving: boolean                    // Сохраняется ли сейчас

```

## 🔄 Поток данных

### При загрузке страницы:
```
useParams() → [courseId, chapterId, lessonId]
           ↓
       Параллельная загрузка:
       ├─ getCourse(courseId)
       ├─ getChapter(courseId, chapterId)
       └─ getLesson(courseId, chapterId, lessonId)
           ↓
       Заполнение form с данными
           ↓
       Показать редактор
```

### При редактировании контента:
```
Пользователь вводит текст в textarea
           ↓
       react-hook-form отслеживает изменения
           ↓
       Если открыта вкладка Preview:
       → form.watch("content") триггерит ре-рендер
       → MarkdownPreview получает новый контент
       → Синхронный предпросмотр
```

### При сохранении:
```
Пользователь нажимает "Сохранить"
           ↓
       Валидация через Zod schema
           ↓
       updateLesson(courseId, chapterId, lessonId, data)
           ↓
       Обновление в Firestore
           ↓
       Успех/Ошибка сообщение
           ↓
       router.refresh() для обновления данных
```

### При публикации:
```
Пользователь нажимает "Опубликовать" / "Скрыть"
           ↓
       publishLesson(courseId, chapterId, lessonId, boolean)
           ↓
       Обновление статуса в Firestore
           ↓
       Обновление UI (badge статуса, кнопка)
           ↓
       Успешное сообщение
```

### Валидация
```typescript
const lessonSchema = z.object({
```

## 🎯 Последовательность работ при расширении

Если нужно расширить Lesson Editor:

1. **Добавить новое поле:**
   - Добавить в `Lesson` интерфейс
   - Добавить в `lessonSchema` Zod schema
   - Добавить `FormField` в редактор
   - Обновить `MarkdownPreview` если нужно

2. **Добавить языковую поддержку кода:**
   - Установить language-{language} пакет
   - highlight.js автоматически подберет

3. **Добавить новый тип урока:**
   - Обновить enum в `lesson.types.ts`
   - Добавить `SelectItem` в форму
   - Добавить обработку в компоненте
