## 🛤️ МАРШРУТЫ И ПУТИ

### Основные маршруты:

| Маршрут | Назначение | Статус |
|---------|-----------|--------|
| `/admin/courses` | 📚 Список всех курсов | ✅ |
| `/admin/courses/new` | ➕ Создание нового курса | ✅ |
| `/admin/courses/[courseId]` | 🎯 Dashboard курса | ✅ |
| `/admin/courses/[courseId]/edit` | 📝 Редактирование курса | ✅ |
| `/admin/courses/[courseId]/chapters` | 📖 Главы курса | ✅ |
| `/admin/courses/[courseId]/chapters/[chapterId]/lessons` | 🎬 Уроки главы | ✨ НОВОЕ |

---

## 🔗 НАВИГАЦИОННЫЕ ПЕРЕХОДЫ

### 1️⃣ Список курсов → Dashboard

```
/admin/courses
    ↓ [Открыть]
/admin/courses/[courseId]
```

**Компонент:** `CoursesManagement`
**Кнопка:** "Открыть"

---

### 2️⃣ Dashboard → Редактирование

```
/admin/courses/[courseId]
    ↓ [Редактировать курс]
/admin/courses/[courseId]/edit
    ↓ [Сохранить]
/admin/courses/[courseId] (Dashboard)
```

**Компонент:** `CourseDashboard`
**Form:** `EditCourseForm`

---

### 3️⃣ Dashboard → Главы

```
/admin/courses/[courseId]
    ↓ [Управление главами]
/admin/courses/[courseId]/chapters
```

**Компонент:** `CourseDashboard`
**Page:** `ChaptersPageContent`

---

### 4️⃣ Главы → Уроки

```
/admin/courses/[courseId]/chapters
    ↓ [Открыть] (на главе)
/admin/courses/[courseId]/chapters/[chapterId]/lessons
```

**Компонент:** `ChaptersPageContent`
**Page:** `LessonsPage` (stub)

---

## 📊 ИНФОРМАЦИОННЫЕ ПОТОКИ

### На списке курсов:
- Название курса
- Описание
- Уровень (badge)
- Время прохождения
- Статус (Published/Draft)
- Кнопка "Открыть"

### На Dashboard курса:
- Название + статус
- Полное описание
- Уровень сложности
- Время прохождения
- URL-идентификатор
- ID курса
- 3 кнопки действий

### На странице глав:
- Название курса в header
- Список глав с:
  - Названием
  - Описанием
  - Статусом публикации
  - Порядком
  - Кнопка "Открыть"
  - Кнопка "Удалить"
- Кнопка "Создать главу"

### На странице уроков:
- Информация о курсе и главе
- Информационная карточка
- Кнопка "Создать урок" (disabled - в разработке)
- Кнопки навигации

## 🎨 UI/UX ЭЛЕМЕНТЫ

### Breadcrumbs:
```
Преподаватель > Курсы > [CourseId] > Главы > [ChapterTitle]
```

### Кнопки:
- ✅ "Открыть" - переход на Dashboard курса
- ✅ "Редактировать курс" - переход на edit
- ✅ "Управление главами" - переход на chapters
- ✅ "Удалить курс" - dialog подтверждения
- ✅ "Открыть" (на главе) - переход на lessons
- ✅ "Создать главу" - dialog создания
- ✅ "Удалить" (глава) - dialog подтверждения

### Badges:
- Published / Draft (курс)
- Published / Draft (глава)
- Уровень сложности (Beginner/Intermediate/Advanced)

### Cards:
- Курс с информацией
- Действия
- Информационная карточка (совет)
- Глава с информацией
- Действия
