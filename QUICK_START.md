# 🚀 WebLab - Quick Start Guide

## За 5 минут до первого урока

### 1️⃣ Запуск приложения

```bash
# Установить зависимости
npm install

# Запустить dev сервер
npm run dev

# Открыть http://localhost:3000
```

---

### 2️⃣ Вход в систему

**Админ:**
```
Email: admin@example.com
Password: 123456
```

**Студент:**
```
Email: student@example.com  
Password: 123456
```

---

### 3️⃣ Создание первого курса (Admin)

1. 📍 Перейти: `/admin/courses`
2. ➕ Нажать "Новый курс"
3. 📝 Заполнить форму:
   - Название: "JavaScript Basics"
   - Описание: "Learn JavaScript from scratch"
   - Уровень: "Beginner"
   - Время: 20 часов

4. 💾 Нажать "Создать"

---

### 4️⃣ Добавление главы

1. 📍 Открыть созданный курс
2. ➕ Нажать "Новая глава"
3. 📝 Заполнить:
   - Название: "Introduction"
   - Описание: "Get started with JavaScript"

4. 💾 Нажать "Создать"

---

### 5️⃣ Создание урока с контентом

1. 📍 Открыть главу
2. ➕ Нажать "Новый урок"
3. 📝 Заполнить:
   - Название: "What is JavaScript?"
   - Описание: "Basics explanation"
   - Тип: "📖 Теория"
   - Время: 15 минут

4. 🎯 Написать контент в Markdown:

```markdown
# What is JavaScript?

JavaScript - это язык программирования для веб-разработки.

## Основные возможности

- Интерактивность на веб-страницах
- Манипуляция DOM
- Асинхронные операции

### Пример кода

```js
const greeting = "Hello, World!";
console.log(greeting);
```

```

5. 💾 Нажать "Сохранить"
6. 🔓 Нажать "Опубликовать урок"

---

### 6️⃣ Просмотр как студент

1. 🔄 Выйти из админ-панели
2. 👤 Авторизоваться как студент
3. 📚 Перейти: `/course`
4. 👀 Вы видите созданный курс!
5. 📖 Кликните на него и читайте контент

## 🗂️ Главные URL пути

### Для админа
```
/admin/courses                    ← Все курсы
/admin/courses/[id]              ← Курс
/admin/courses/[id]/chapters      ← Главы
/admin/courses/[id]/chapters/[id]/lessons/[id]  ← Редактор урока
```

### Для студента
```
/course                           ← Все курсы
/course/[id]                      ← Главы курса
/course/[id]/chapters/[id]        ← Уроки главы
/chapter/[id]                     ← Чтение урока
```

## ⚡ Советы по использованию

✅ **Публикуйте контент**
- Без публикации студенты не видят уроки
- Используйте кнопку "Опубликовать урок"

✅ **Используйте Markdown**
- Красиво оформляйте текст
- Добавляйте примеры кода
- Делайте таблицы

✅ **Тестируйте на Preview**
- Переключайтесь на вкладку Preview
- Видите как выглядит для студентов
- Проверьте все форматирование

✅ **Используйте Markdown справку**
- На странице редактора внизу
- Примеры готовой синтаксиса
- Скопируйте и используйте
