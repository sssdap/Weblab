# Обновление: Добавлены кнопки публикации для глав и уроков

## Что было добавлено

### ✅ Функциональность публикации глав

На странице управления главами курса теперь есть кнопки для быстрой публикации:

- **Зелёная кнопка "Опубликовать"** — выложить главу для студентов
- **Красная кнопка "Скрыть"** — скрыть главу из курса
- Статус обновляется мгновенно
- Кнопки появляются на каждой карточке главы

### ✅ Функциональность публикации уроков

На странице управления уроками главы теперь есть кнопки для быстрой публикации:

- **Зелёная кнопка "Опубликовать"** — выложить урок для студентов
- **Красная кнопка "Скрыть"** — скрыть урок из главы
- Статус обновляется мгновенно
- Кнопки появляются на каждой карточке урока

### ✅ Индикаторы статуса

На всех карточках есть визуальные индикаторы:

- **[Опубликована]** — зелёная иконка (видна студентам)
- **[Черновик]** — серая иконка (скрыта от студентов)

## Использование

### Шаг 1: Опубликовать главу

1. Откройте админ-панель
2. Перейдите на страницу курса (Главы)
3. Найдите главу
4. Нажмите зелёную кнопку **"Опубликовать"**
5. ✅ Статус изменился на **"Опубликована"**

### Шаг 2: Опубликовать уроки в главе

1. На странице Глав нажмите **"Открыть"** на нужной главе
2. Перейдите в раздел **"Уроки"**
3. Для каждого урока нажмите **"Опубликовать"**
4. ✅ Статус изменился на **"Опубликован"**

### Шаг 3: Проверить в приложении

1. Откройте приложение (студенческий вид)
2. Перейдите в `/course`
3. Должны видеть опубликованные курсы
4. Откройте курс
5. Должны видеть опубликованные главы
6. Откройте главу
7. Должны видеть опубликованные уроки ✅

## Логирование

В консоли браузера (F12 → Console) будут логи:

```
[CHAPTERS PAGE] Publishing chapter: chapter-123
[CHAPTERS PAGE] Chapter publish state toggled successfully

[LESSONS PAGE] Publishing lesson: lesson-456
[LESSONS PAGE] Lesson publish state toggled successfully
```

Это помогает отследить что происходит во время публикации.

---

## Обработка ошибок

Если что-то пошло не так:

1. **В консоли браузера** будут ошибки:
   ```
   [CHAPTERS PAGE] Error toggling publish: Error message...
   ```

2. **В интерфейсе** появится красное сообщение об ошибке

3. **Кнопка разблокируется** и можно будет попробовать снова
