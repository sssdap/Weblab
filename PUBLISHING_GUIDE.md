# Публикация Глав и Уроков - Руководство

## Обзор

Теперь в админ-панели добавлены кнопки для быстрой публикации (выкладки) глав и уроков. Это позволяет управлять видимостью контента для студентов.

---

## Структура системы публикации

### Статусы публикации

Каждый элемент может быть в одном из двух состояний:

- **🟢 Опубликован** (Published) — видим студентам
- **🔒 Черновик** (Draft) — скрыт от студентов

### Уровни публикации

Публикация работает на **трёх уровнях**:

1. **Курс** — если не опубликован, студенты не видят его
2. **Глава** — должна быть опубликована, чтобы студенты видели её в курсе
3. **Урок** — должен быть опубликован, чтобы появиться в главе

**Все три уровня должны быть опубликованы**, чтобы студент мог увидеть урок!

---

## Как опубликовать контент

### Публикация глав

1. Откройте курс в админ-панели
2. Перейдите на вкладку **Главы** или нажмите на курс
3. Найдите главу, которую хотите опубликовать
4. На карточке главы нажмите кнопку **Опубликовать** (зелёная, с иконкой 👁️)
5. Статус на карточке изменится на **Опубликована**

**Статусы кнопки:**
- 🟢 **Опубликовать** (зелёная) — глава в черновике, нажмите чтобы выложить
- 🔴 **Скрыть** (красная) — глава опубликована, нажмите чтобы спрятать

### Публикация уроков

1. В админ-панели откройте курс → Главы
2. Выберите главу, нажав **Открыть**
3. Перейдите в раздел **Уроки**
4. Найдите урок, который хотите опубликовать
5. На карточке урока нажмите кнопку **Опубликовать** (зелёная, с иконкой 👁️)
6. Статус изменится на **Опубликован**

**Статусы кнопки:**
- 🟢 **Опубликовать** (зелёная) — урок в черновике, нажмите чтобы выложить
- 🔴 **Скрыть** (красная) — урок опубликован, нажмите чтобы спрятать

## Полный путь публикации

### Шаг за шагом

```
1. Админ-панель → Курсы
   ↓
2. Выберите курс (убедитесь что он опубликован)
   ↓
3. Нажмите "Главы" или откройте курс
   ↓
4. 🟢 Опубликуйте главу кнопкой "Опубликовать"
   ↓
5. Нажмите "Открыть" на главе
   ↓
6. Нажмите на "Уроки"
   ↓
7. 🟢 Опубликуйте каждый урок кнопкой "Опубликовать"
   ↓
8. ✅ Готово! Студенты видят урок в приложении
```

## Специальные кнопки в редакторе урока

При редактировании урока есть **отдельная карточка "Статус урока"**:

```
┌──────────────────────────────────────────┐
│ 📋 Статус урока                         │
│ Управление видимостью урока для студ.   │
│                                          │
│ [Опубликована] (или [Черновик])          │
│                                          │
│ Этот урок видим для студентов...         │
│                                          │
│ [Опубликовать урок] или [Скрыть урок]   │
└──────────────────────────────────────────┘
```

Эта кнопка дублирует функцию, но позволяет быстро менять статус прямо во время редактирования.

## Проверка результата

### Как убедиться что всё опубликовано

1. **В админ-панели:**
   - На карточке должен быть статус **"Опубликована"** / **"Опубликован"**
   - Кнопка должна быть **красной** (означает, что контент виден)

2. **В приложении для студентов:**
   - Откройте `/course`
   - Должны видеть курс
   - Откройте курс
   - Должны видеть опубликованные главы
   - Откройте главу
   - Должны видеть опубликованные уроки
   - Откройте урок
   - Должны видеть его содержимое

### Если студент не видит контент

**Проверьте в этом порядке:**

1. ❓ **Курс опубликован?**
   - Admin → Courses → проверьте статус

2. ❓ **Глава опубликована?**
   - Admin → Courses → выберите курс → Chapters
   - Нажмите на главу, проверьте статус

3. ❓ **Урок опубликован?**
   - Admin → Courses → выберите курс → Chapters → выберите главу → Lessons
   - Нажмите на урок, проверьте статус

4. ❓ **Правильный ли индекс Firestore?**
   - Убедитесь что у вас созданы все 3 индекса (courses, chapters, lessons)
   - Смотрите: `FIRESTORE_INDEXES.md`

## Быстрые советы

### Советы по организации

- 💡 **Сначала создайте всё**, потом опубликуйте
  - Создавайте черновики уроков
  - Редактируйте и проверяйте
  - Когда готово — опубликуйте

- 💡 **Используйте "Черновик" для тестирования**
  - Создавайте новые уроки в черновике
  - Проверяйте содержимое
  - Потом опубликуйте

- 💡 **Можно скрывать контент**
  - Если нужно отредактировать урок
  - Скройте его (нажмите кнопку "Скрыть")
  - Отредактируйте
  - Опубликуйте снова

### Частые ошибки

❌ **"Глава не видна в приложении"**
- Проверьте что глава **опубликована** ✅
- Проверьте что хотя бы один урок **опубликован** ✅
- Проверьте индексы Firestore

❌ **"Урок не видна в главе"**
- Проверьте что урок **опубликован** ✅
- Проверьте что глава **опубликована** ✅
- Очистите кэш браузера (Ctrl+Shift+Del)

❌ **"Кнопка не работает"**
- Дождитесь загрузки (крутящаяся иконка)
- Обновите страницу
- Проверьте консоль браузера (F12) на ошибки
