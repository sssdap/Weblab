# 🎯 Course Dashboard - Краткая инструкция

## 📍 Что создано?

**Course Dashboard** - центральная страница управления курсом в админке.

```
/admin/courses/[courseId]  ← 🎯 ВЫ ЗДЕСЬ (Dashboard)
```

## 🎮 Как это работает?

### Вход на Dashboard

**Из списка курсов:**
```
/admin/courses (список)
    ↓ клик "Открыть"
/admin/courses/my-course-123 (Dashboard)
```

### Три основных действия

#### 1️⃣ Редактировать курс
```
Нажать "Редактировать курс"
    ↓
/admin/courses/[courseId]/edit
    ↓ Сохранить
/admin/courses/[courseId] (назад на Dashboard)
```

#### 2️⃣ Управление главами
```
Нажать "Управление главами"
    ↓
/admin/courses/[courseId]/chapters
```

#### 3️⃣ Удалить курс
```
Нажать "Удалить курс"
    ↓ Dialog подтверждения
"Удалить" или "Отмена"
    ↓ если Удалить
/admin/courses (список)
```

## 🚀 Быстрый старт

### Чтобы попасть на Dashboard:

```
1. Откройте админку
2. Перейдите в "Курсы"
3. Нажмите "Открыть" на любом курсе
4. Вы на Dashboard! ✅
```

### Чтобы редактировать курс:

```
На Dashboard → нажмите "Редактировать курс"
    ↓
Измените данные
    ↓
Нажмите "Сохранить изменения"
    ↓
Вернётесь на Dashboard ✅
```

### Чтобы удалить курс:

```
На Dashboard → нажмите "Удалить курс"
    ↓
Подтвердите в диалоге
    ↓
Курс удалён, редирект на список ✅
```

## 🔄 Полный поток

```
Админка
  ↓
Курсы (/admin/courses)
  ↓ [Открыть]
Dashboard (/admin/courses/[courseId]) ⭐ ВЫ ЗДЕСЬ
  ├─→ [Редактировать] → Edit (/edit)
  │                      ├→ [Сохранить] → Dashboard
  │                      └→ [Отмена] → Dashboard
  ├─→ [Главы] → Chapters (/chapters)
  └─→ [Удалить] → Dialog
                   ├→ [Удалить] → Список курсов
                   └→ [Отмена] → Dashboard
```

## 📊 Показываемая информация

| Элемент | Описание |
|---------|---------|
| Название | Название курса с бейджем статуса |
| Описание | Полное описание курса |
| Уровень | Beginner / Intermediate / Advanced |
| Время | Примерное время прохождения |
| Slug | URL-идентификатор курса |
| ID | Уникальный идентификатор |

## 🎨 UI Компоненты

- **Card** - контейнеры информации
- **Button** - кнопки действий (Edit, Chapters, Delete)
- **Badge** - статусы (Published/Draft, уровень)
- **AlertDialog** - подтверждение удаления
- **Alert** - сообщения об ошибках
- **Loader** - индикатор загрузки

## ⚡ Особенности

✅ Loading state во время загрузки
✅ Error handling при ошибках
✅ Подтверждение перед удалением
✅ Breadcrumbs для навигации
✅ Responsive design (мобильный/планшет/ПК)
✅ Dark mode поддержка
✅ TypeScript типизация

## 🔒 Доступ

- **Только для админов** (role === "admin")
- Проверка в middleware
- При отсутствии доступа → редирект на `/dashboard`
